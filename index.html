Projects that I have worked on

  • 1. Data Migration and user data analysis

    Some details of the project

    The project consisted of two parts – one, to migrate Data from one Customer management platform to a new one and second to perform analysis on the user specific data.
    My client was an online business owner and offered courses over multiple educational platforms. Naturally, they had volumes of user information from multiple sources and they also had user statistics data, on which we planned to perform data analysis to get more insights that would help the client with their business strategies.

Data Migration: Since, the client had user information in CSV files from multiple sources/platforms and wanted to move to a new management system, this project aimed on data cleaning, restructuring and then combining the files to get them ready for migration.

Data Analysis: The data for analysis were sourced from YouTube studio and Wordpress administrator accounts and contained information like the countries from where the course videos were being viewed, the no. of hits on each video, traffic sources, etc. The objective of this part of the project was to perform data cleaning and then create dashboards after analysis.
